# SharedStarrySky
This project aims to build a global platform that connects people to professional telescopes around the world. Users will be able to reserve observation time, remotely control telescopes, and capture astrophotography through a simple interface on devices such as iPhone, iPad, and other connected platforms. This project connects telescope owners with remote users who want high-quality astronomical observations. Because location and timing are critical for capturing clear images, the platform enables people without telescopes to access premium equipment remotely. It reduces idle time for expensive telescopes while generating income for owners and expanding affordable access to astrophotography.
